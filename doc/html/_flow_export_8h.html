_LIB_INC_RIP_API_ void globalFlowInit ( _In_ WCHAR *  modulesNames,
_In_ DWORD  numberOfModule 
)

Init flow in the current process, but do not start it. If you are calling this function manually you should consider calling debugCurrentThread on the main thread of the process.

Parameters
[in]modulesNamestable of all module path which make 500 byte length. It length should be equal to numberOfModule * 500 at least.
[in]numberOfModulenumber of module. Between 0 and 10 or it fails.
